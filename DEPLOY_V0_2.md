# Déploiement v0.2

1. Décompresser le ZIP.
2. Dans GitHub : Add file → Upload files.
3. Glisser tout le contenu intérieur à la racine du dépôt.
4. Accepter le remplacement des fichiers existants.
5. Commit : `Add Anatole Cockpit TSX60 v0.2`.
6. Vérifier Render : l’API doit redéployer et `/docs` doit afficher `GET /api/v1/market/cockpit`.
7. Vérifier Vercel : le nouveau déploiement doit être Ready.
8. Ouvrir `/cockpit`, puis cliquer sur une tuile pour ouvrir Focus.

Aucune nouvelle variable d’environnement n’est requise.
